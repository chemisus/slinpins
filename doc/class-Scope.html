public string[]
# keys( ReflectionParameter[] $params, string[] $annotations = array(), string[] $injections = array() )

Returns an array containing the keys that should be used to inject into a method.

Returns an array containing the keys that should be used to inject into a method.

The keys to be used can be supplied from the following sources, and will be set in the following order: 1. parameters 2. annotations 3. injections
